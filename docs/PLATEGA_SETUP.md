# Platega.io — пошаговая регистрация

## Зачем

Platega — российский платёжный шлюз для приёма оплаты от клиентов и **вывода в USDT TRC-20 без банковских проблем**. Для нас — основной канал монетизации.

## Регистрация

### Шаг 1: Перейти на сайт
Открой https://platega.io в браузере.

### Шаг 2: Кнопка "Регистрация" (правый верхний угол)
На экране увидишь:
```
┌──────────────────────────────────────────┐
│ Platega.io          [Войти] [Регистрация]│
├──────────────────────────────────────────┤
│                                          │
│      Платёжный шлюз нового поколения     │
│                                          │
└──────────────────────────────────────────┘
```
Нажми **Регистрация**.

### Шаг 3: Форма
Заполни:
- **Email** (любой, лучше отдельный для бизнеса)
- **Пароль** (сложный, 12+ символов)
- **Подтверждение пароля**
- Согласие с офертой → галочка
- Нажми **Зарегистрироваться**

На email придёт письмо с подтверждением. Кликни по ссылке в письме.

### Шаг 4: Подтверждение личности (KYC)
После логина в личный кабинет:
1. Перейди в раздел **"Настройки"** → **"Верификация"**
2. Загрузи:
   - Фото/скан **паспорта** (первая страница)
   - Селфи с паспортом в руке
3. Заполни ИНН (если ИП/ООО) или оставь пустым (физлицо)
4. Подожди 1-24 часа (обычно 30 мин)

### Шаг 5: Создание магазина
В личном кабинете:
1. **"Магазины"** → **"Создать магазин"**
2. Заполни:
   - **Название**: "Buildo Plati Bot" (или своё)
   - **URL уведомлений**: `http://108.165.164.85:8080/platega/callback`
   - **Валюта по умолчанию**: RUB
   - **Способ выплат**: **USDT TRC-20** (в крипту)
   - **Адрес кошелька USDT TRC-20**: твой TRC-20 адрес (например из @CryptoBot)
3. Нажми **Создать**

### Шаг 6: Получение API ключей
После создания магазина:
1. Перейди в **"Магазины"** → выбери свой → **"API"**
2. Увидишь:
   ```
   Merchant ID: 12345
   API Key: xxxxxxxxxxxxxxxxxxxxxxxx
   ```
3. **Скопируй оба значения**

### Шаг 7: Вписать в .env на сервере
Подключись к серверу и отредактируй `.env`:
```bash
ssh root@108.165.164.85
nano ~/Projects/buildo-plati-bot/.env
```
Найди строки:
```
PLATEGA_API_KEY=
PLATEGA_MERCHANT_ID=
```
Замени пустые значения на реальные. Сохрани (Ctrl+O, Enter, Ctrl+X).

### Шаг 8: Перезапустить бота
```bash
cd ~/Projects/buildo-plati-bot
docker compose restart
docker compose logs --tail=20
```
В логах должно появиться: `Platega webhook configured for merchant 12345`.

## Готово!

Теперь бот принимает оплату через Platega, клиенты платят рублями/СБП/картой, деньги приходят на твой USDT TRC-20 кошелёк.

## Тест

Создай тестовый заказ через бота → перейди по ссылке оплаты → проверь что webhook вызвался (`docker compose logs | grep platega`).
